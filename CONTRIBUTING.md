# Contributing to NomaTune

Thanks for your interest in contributing! 🎵

## 🐛 Reporting Bugs

Please open an [issue](https://github.com/Shahdullah/NomaTune/issues) with:
- Your device & Android version
- NomaTune version
- Steps to reproduce
- Expected vs actual behavior
- Logs / screenshots if possible

## ✨ Feature Requests

Open an issue with the `[Feature Request]` prefix and describe:
- What you want to add
- Why it would be useful
- Mockups or examples (optional)

## 🔧 Pull Requests

1. Fork the repo
2. Create a feature branch: `git checkout -b feat/your-feature`
3. Make your changes
4. Test on a real device
5. Open a PR with a clear description

### Code Style
- Follow Kotlin official style guide
- Use 4-space indentation
- Write clear commit messages

## 🌍 Translations

Translations are welcome! Edit the appropriate `strings.xml` file in `app/src/main/res/values-XX/`.
